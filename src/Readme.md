## Doc Driven Development

### 1. Remplacer un aliment
#### Pre-requis
Un clavier standard.
#### Explications
L'utilisateur sélectionne dans la première liste de choix :
> Quel aliment souhaitez-vous remplacer ?

Le programme pose les questions suivantes à l'utilisateur et ce dernier sélectionne les réponses :
1. Sélectionnez la catégorie. [Plusieurs propositions associées à un chiffre. L'utilisateur entre le chiffre correspondant et appuie sur entrée]
2. Sélectionnez l'aliment. [Plusieurs propositions associées à un chiffre. L'utilisateur entre le chiffre correspondant à l'aliment choisi et appuie sur entrée]
3. Le programme propose un substitut, sa description, un magasin ou l'acheter (le cas échéant) et un lien vers la page d'Open Food Facts concernant cet aliment.
4. L'utilisateur a alors la possibilité d'enregistrer le résultat dans la base de données.
